iPost.crx 掛號快查 (Google Chrome Extension)
=========
透過網路郵局簡易查詢掛號信件及包裹狀態
-------------

本套件提供最簡潔、最快速的查詢介面，查詢中華郵政國內快捷/掛號/包裹。

查詢即直接連接至中華郵政公司網路郵局，
非透過第三方伺服器，確保結果一致、可靠。

Now available on [chrome web store (掛號快查)](
